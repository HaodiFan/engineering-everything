# Wrapper Migration: v0.2.0 to v0.3.0

## Summary

Upgrade EvoZeus-wrapper from `v0.2.0` to `v0.3.0` for `engineering-everything`.

## From wrapper version

## To wrapper version

## Reason

EvoZeus-wrapper `v0.3.0` requires wrapped target Skills to run a status check before their main chain. The check covers the current Skill release, wrapper harness version, source contract, and remediation order.
